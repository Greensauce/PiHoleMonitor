# Screenshots

This directory contains screenshots for the Pi-hole Plasma Widget.

## Needed Screenshots

Please add the following screenshots:

1. **widget-example.png** - Main widget in panel showing status
2. **widget-expanded.png** - Expanded view with all controls
3. **widget-disabled.png** - Widget showing paused state with countdown
4. **widget-config.png** - Configuration dialog
5. **widget-disable-options.png** - Inline disable duration options

## Screenshot Guidelines

- Take screenshots on a dark theme (matches most KDE users)
- Ensure widget is clearly visible
- Show realistic data (real Pi-hole connection)
- Resolution: At least 1920x1080
- Format: PNG with transparency if possible
- File size: Keep under 500KB each

## How to Add Screenshots

1. Take screenshots according to guidelines
2. Save to this directory with appropriate names
3. Update README.md to reference them
4. Commit and push

Example README.md usage:
```markdown
![Widget in Panel](docs/screenshots/widget-example.png)
```
